# CodeKitchen

- Step-by-step algorithm gameplay
- Visual representation of data structures
- Efficiency and complexity feedback
- Friendly guidance and mistake explanation

## Game Modes
### Algorithm Mode
Players solve algorithm problems by performing actions that mirror real algorithm steps.

### System Design Mode (Planned)
Players visually design systems, simulate load, and learn tradeoffs.
